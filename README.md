# Pulse — каталог интернет-магазина медтех-устройств

Тестовое задание: секция каталога интернет-магазина, 6 карточек товара.
Тематика — премиальные носимые устройства для мониторинга здоровья
(смарт-кольцо, ЭКГ-монитор, пульсоксиметр и т.д.).

**[Открыть демо](./index.html)** — просто откройте файл в браузере, сеть не нужна.

## Стек

Чистый HTML + CSS, без UI-китов и фреймворков. Один самодостаточный
`index.html`: шрифты и фотографии товаров зашиты внутрь base64, страница
открывается двойным кликом без сервера и работает офлайн.

- **Вёрстка** — Flexbox + CSS Grid, адаптив на 7 брейкпоинтах (320–1600px)
- **Дизайн-система** — CSS custom properties: 2 шрифта, 3 радиуса,
  3 тайминга, вся сетка отступов и кеглей кратна 4px
- **Интерактив** — hover-состояния на карточках (кросс-фейд «товар → на теле»),
  фильтрация по категориям, `View Transitions API` при смене фильтра
  с graceful fallback
- **Анимации** — нативные scroll-driven animations (`animation-timeline: view()`),
  без единой строчки анимационного JS
- **Доступность** — `:focus-visible` на всех интерактивных элементах,
  `aria-live` на счётчике результатов, `prefers-reduced-motion` учтён
  в четырёх местах
